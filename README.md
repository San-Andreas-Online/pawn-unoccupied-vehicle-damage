## Introduction

This script was originally made by KingHual, and is now maintained by San Andreas Online. It adds some new functionalities to detect and handle damage and collisions for unoccupied vehicles in SA-MP. The script includes functions to retrieve vehicle sizes, calculate directional speed differences, and callbacks for when unoccupied vehicles are damaged or collide with other entities.  
You can find the full list of functions and their documentation in [WIKI.md](docs/WIKI.md).
